# themeforge

a tool for generating accessible design systems from a single color.

it's built for white-label platforms where you need to spin up a professional brand identity without manual design work. the engine handles color math, wcag compliance, and semantic token mapping.

## features
- **automated generation:** generates full themes from a primary hex.
- **accessibility first:** real-time wcag 2.1 contrast validation.
- **live preview:** test themes against e-commerce and dashboard layouts.
- **developer export:** get tailwind configs, css variables, or raw design tokens.

## stack
- react 19 / vite
- tailwind 4
- typescript

## usage
```bash
npm i
npm run dev
```
