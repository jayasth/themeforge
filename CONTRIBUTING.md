# contributing to themeforge

if you're helping out, keep it simple.

- **no bloat:** keep the code lean.
- **technical focus:** we build tools for developers and white-label platforms.
- **modern stack:** stay within react 19 and tailwind 4.
- **typescript:** no any.

## workflow
1. fork it.
2. make your changes.
3. submit a pr with a clear description of what changed.
